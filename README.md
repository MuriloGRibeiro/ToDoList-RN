# ToDoList-RN

To Do List feito com React Native

Instalar Dependências

1- Abra o VSCODE

2- Abra um terminal dentro da pasta do projeto

npm -install

Após digitar esse comando, será criado a pasta "node_modules", no qual as dependências para rodar o projeto estarão lá.
Caso precise enviar o projeto, apague a pasta "node_modules" e envie zipado, com o comando pode ser feito o download novamente.

3- Para executar:

npm -start
